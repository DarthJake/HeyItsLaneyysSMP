# Hey Its Laney's SMP

This is the git repo to manage the [HeyItsLaneyysSMP](https://www.curseforge.com/minecraft/modpacks/heyitslaneyyssmp) modpack.

This modpack uses [packwiz](https://packwiz.infra.link/) to manage the development of the pack.
It is not required to install packwiz if all you want to do is run the pack. Only if you'll be doing development or updating the pack.
